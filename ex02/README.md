# 第2回
## 超高機能電卓（ex02/calc.py）
### 追加機能
- 四則演算：＋以外の四則演算をする
- クリア(BC)：entryに入力されている数字，数式の1文字をdeleteする
- オールクリア(C)：entryに入力されている数字，数式の文字列全体をdeleteする
- パーセント：entryに入力されている数値を、%表示にする
- 小数点の入力：entryに入力されている数値に小数点表示にする
- 配置変更：0の数値を数値の一番下の真ん中に来るように配置
- 平方根ボタン：entryに入力されている数字を、√の数値でする
